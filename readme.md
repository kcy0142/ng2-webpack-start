# 설명
>1. angular-cli를 사용하지 않고 순수 angular2를 사용해서 webpack 구현
>2. angular-cli로 구현하면 내부적으로 ng eject로 webpack.config.js파일을 추출가능
>3. http://kcy0142.blog.me/220929154769 에 설명되어 있음  

# Install
``` bash
$ npm install
```

# dev-server
``` bash
$ npm start
```

# Build
``` bash
# build
$ npm build
# minify and build
$ npm build:prod
```
